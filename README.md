# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

A comprehensive Power BI dashboard analyzing **18 seasons of IPL data** (2008–2025), featuring season-wise stats, champion & runner-up, Orange/Purple Cap holders, Points Table, and more.

## ✨ Features

- 🗓️ **Season Filter** — Switch between any IPL season (2008–2025)
- 🏅 **Champion & Runner-Up** cards per season
- 🟠 **Orange Cap** & 🟣 **Purple Cap** holder stats
- 🏏 **Total 6s, 4s, Matches, Teams, Centuries, Half Centuries, Venues**
- 📋 **Points Table** with Pld / Won / Lost / NR / Tie / Total Points
- 👤 **Top Fours & Top Sixes** player cards with photos

---

## 🗂️ Data Sources

| Table | Description |
|---|---|
| `ball_by_ball_data` | Ball-by-ball match data |
| `ipl_matches_data` | Match-level results |
| `players-data-updated` | Player profiles & stats |
| `teams_data` | Team information |

---

## 🛠️ Tools Used

- **Power BI Desktop**
- **DAX** for calculated measures
- **Power Query** for data transformation

---

## 🚀 How to Use

1. Clone or download this repository
2. Open `IPL 2008-2025.pbix` in Power BI Desktop
3. Use the **Season slicer** to explore any IPL year
4. Interact with visuals to drill down into stats
