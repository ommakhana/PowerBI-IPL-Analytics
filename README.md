# 🏏 IPL Analytics Dashboard (2008–2025) — Power BI Project

An interactive **Power BI dashboard** covering **17 seasons of the Indian Premier League (IPL)** — from 2008 to 2025.

Explore team performance, player stats, match outcomes, toss trends, and more.

## 🚀 Getting Started

1. Open **Power BI Desktop**
2. Load `IPL 2008-2025.pbit`
3. Connect data files if prompted
4. Click **Refresh**
5. Explore the dashboard 🎉
